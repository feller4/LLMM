HTML: Las Etiquetas Clave para Empezar

El HTML es como el esqueleto de tu web. Aquí tienes las etiquetas más útiles para organizar el contenido.

Índice Rápido

  • La etiqueta <div> (Contenedor)
  • La etiqueta <a> (Enlace)
  • La etiqueta <ul> (Lista)
  • La etiqueta <table> (Tabla)
  • La etiqueta <details> (Mostrar/Ocultar)

1. La etiqueta <div>

Es el contenedor más usado. Sirve para agrupar varias cosas (texto, imágenes) y luego ponerles color o moverlas juntas con CSS.

Ejemplo Sencillo

<div>
    <h3>Aquí va mi contenido</h3>
    <p>Esta caja lo agrupa todo.</p>
</div>

2. La etiqueta <a>

Crea enlaces. Es lo que hace que puedas ir de una página a otra. El atributo `href` dice a dónde ir.

Ejemplo Sencillo

<a href="https://www.google.com">
    Ir a Google
</a>

Ver más en W3C (MDN)

3. La etiqueta <ul>

Se usa para hacer listas donde el orden no importa (listas desordenadas). Cada punto de la lista es una etiqueta `<li>`.

Ejemplo Sencillo

<ul>
    <li>Primer Punto</li>
    <li>Segundo Punto</li>
</ul>

Se ve así:

Ver más en W3C (MDN)

4. La etiqueta <table>

Sirve para mostrar datos en filas y columnas, como en una hoja de cálculo. Se usa `<tr>` para las filas y `<td>` para las celdas.

Ejemplo Sencillo

<table border="1">
    <tr><td>Celda 1</td><td>Celda 2</td></tr>
    <tr><td>Celda 3</td><td>Celda 4</td></tr>
</table>

Se ve así:

Ver más en W3C (MDN)

5. La etiqueta <details>

Permite ocultar un texto que el usuario puede ver haciendo clic. Es muy útil para preguntas frecuentes. El texto que se ve es el que pones en `<summary>`.

Ejemplo Sencillo

<details>
    <summary>Haz clic para ver la respuesta</summary>
    <p>Aquí está el texto oculto.</p>
</details>
