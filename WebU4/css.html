CSS: Cómo Darle Estilo

El CSS son las reglas de estilo. Con esto le dices a tus elementos de HTML qué colores, tamaños y posiciones deben tener.

Índice Rápido

  • display: flex (Organizar en fila)
  • grid-template-columns (Cuadrículas)
  • border-radius (Esquinas Redondas)
  • box-shadow (Sombras)
  • transition (Movimiento Suave)

1. La Propiedad `display: flex`

Es la forma más fácil de organizar elementos en fila (o columna) y que se alineen solos.

Ejemplo Sencillo

.contenedor {
    display: flex;
    justify-content: center;
}

Resultado en el Navegador

Ver más en W3C (MDN)

2. La Propiedad `grid-template-columns`

Te permite crear una cuadrícula, como una tabla, para organizar secciones complejas. Esta propiedad define el ancho de las columnas.

Ejemplo Sencillo

.contenedor-grid {
    display: grid;
    grid-template-columns: 1fr 1fr;
}

Resultado en el Navegador

Ver más en W3C (MDN)

3. La Propiedad `border-radius`

Hace que las esquinas de un elemento sean redondas en lugar de cuadradas.

Ejemplo Sencillo

.mi-caja {
    border-radius: 10px;
}

Resultado en el Navegador

Ver más en W3C (MDN)

4. La Propiedad `box-shadow`

Añade una sombra a una caja para darle profundidad. Los números controlan qué tan lejos y qué tan difusa es la sombra.

Ejemplo Sencillo

.tarjeta {
    box-shadow: 2px 2px 5px gray;
}

Resultado en el Navegador

Ver más en W3C (MDN)

5. La Propiedad `transition`

Sirve para que los cambios de estilo (como el color o el tamaño) no sean instantáneos, sino que se muevan suavemente durante un tiempo.

Ejemplo Sencillo

.boton {
    background-color: blue;
    transition: 0.5s;
}
.boton:hover {
    background-color: red;
}

Resultado en el Navegador

Ver más en W3C (MDN)
