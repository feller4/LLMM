SVG: Dibujos para la Web

El SVG sirve para hacer gráficos y dibujos con código. Lo bueno es que se ven bien aunque los hagas gigantes.

Índice Rápido

  • El Elemento <circle> (Círculo)
  • El Elemento <rect> (Rectángulo)
  • El Elemento <path> (Forma Libre)
  • El Elemento <text> (Texto)
  • El Elemento <g> (Agrupar)

1. El Elemento <circle>

Dibuja un círculo. Necesita `cx` y `cy` (dónde está el centro) y `r` (qué tan grande es el radio).

Ejemplo Sencillo

<svg width="100" height="100">
    <circle cx="50" cy="50" r="40" fill="red" />
</svg>

2. El Elemento <rect>

Dibuja un rectángulo. Necesita `width` (ancho) y `height` (alto).

Ejemplo Sencillo

<svg width="120" height="80">
    <rect width="100" height="60" fill="green" />
</svg>

Se ve así:

Ver más en W3C (MDN)

3. El Elemento <path>

Dibuja cualquier forma que imagines. Usa el atributo `d` con comandos especiales (M=mover, L=línea, C=curva). Es un poco más avanzado.

Ejemplo Sencillo (Una Línea)

<svg width="100" height="100">
    <path d="M10 10 L90 90" stroke="black" />
</svg>

4. El Elemento <text>

Para poner texto dentro del dibujo SVG. Usas `x` y `y` para ubicarlo.

Ejemplo Sencillo

<svg width="200" height="50">
    <text x="10" y="30" fill="purple">Hola Texto</text>
</svg>

Se ve así:

Ver más en W3C (MDN)

5. El Elemento <g>

Sirve para agrupar formas. Si mueves el `<g>`, todas las formas que estén dentro se mueven juntas.

Ejemplo Sencillo

<svg width="150" height="80">
    <g transform="translate(10, 10)">
        <rect width="30" height="30" fill="gray" />
    </g>
</svg>
